## About The Project

Naurex Core is a premium system optimization utility developed for Windows. Designed specifically for power users, gamers, and performance enthusiasts, the software operates directly at the OS core to disable superfluous background services, eliminate telemetry bottlenecks, and optimize hardware scheduling.

Unlike aggressive, generic optimizers that often break system functionality, Naurex Core has been strictly audited to maintain the integrity of modern applications and native Windows features. It safely debloats the operating system without compromising stability.

Featuring nearly 100 deep-level optimizations focused on reducing network latency, maximizing disk I/O speeds, and improving memory management, the software is built around a strict "Read-Before-Write" security system. It creates precise, instantaneous backups of your system's registry configurations before applying any changes, ensuring you can safely revert modifications at any time with a single click.

## Key Capabilities

* **Completely Safe & Reversible:** Features a built-in backup engine that captures the exact state of your system before applying tweaks. This allows for instant and reliable restoration if you ever wish to revert to your original settings.
* **True Performance Focus:** Enables hardware-accelerated GPU scheduling (HAGS), forces SSD TRIM for sustained drive speeds, isolates Windows Explorer processes to prevent system-wide freezes, and unlocks 100% network bandwidth by removing native OS throttling.
* **Built for Gamers:** Disables mouse acceleration to force true raw input for muscle memory accuracy, turns off multiplane overlay (MPO) to prevent visual stuttering, and disables background capture services for minimal input latency and maximum frame rates.
* **Deep System Cleanup:** Flushes temporary data, clears massive cached files, and optimizes session pool sizes to free up physical storage and RAM natively.
* **Premium User Experience:** Clean, ad-free, borderless dark-themed interface designed to be straightforward, fast, and completely non-obtrusive.

## How to Use

Naurex Core is designed to be direct and user-friendly. 

1. **Run as Administrator:** Naurex Core requires administrative privileges to interact with the Windows core, services, and registry.
2. **Create a Backup:** Before applying any optimizations, click the **CREATE BACKUP** button. The software will scan your current environment and generate a secure backup file. Keep this file safe.
3. **Select Optimizations:** Browse the list and check the boxes for the tweaks you wish to apply. You can use standard Windows selection shortcuts (Shift+Click for multiple items).
4. **Apply:** Click the **APPLY SELECTED** button. The engine will silently apply all chosen modifications.
5. **Restart:** A system reboot is highly recommended after the process finishes to ensure all core changes take effect.

## Restoring the System

If you need to undo the changes made by Naurex Core:
1. Open the application.
2. Click the **RESTORE** button.
3. Select the backup file you created previously.
4. The engine will safely rewrite the original values back into your system. Restart your computer to finish the restoration.
